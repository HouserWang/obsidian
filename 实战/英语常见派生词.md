| 特征           | 本土英语（Anglo-Saxon/Germanic）        | 拉丁/法语来源（Latinate）                            | 希腊来源（Greek）                        |
| ------------ | --------------------------------- | -------------------------------------------- | ---------------------------------- |
| **词长**       | 通常短小（1-2音节）                       | 较长（多音节）                                      | 较长，常技术化                            |
| **含义类型**     | 具体、日常、基本事物/动作（concrete）           | 抽象、正式、学术（abstract）                           | 科学、医学、哲学、技术                        |
| **语域**       | 口语、日常、核心词汇（最常见1000词80%是本土）        | 正式、书面、法律/政府/宗教                               | 专业术语（biology, psychology）          |
| **常见声音/辅音簇** | 硬辅音如 sk, ng, ar, th（但th也见希腊）      | con-, re-, pre-, trans-, -tion, -ment        | ph (=f), ch (=k), ps, y (作为元音), th |
| **常见前缀**     | un-, over-, under-, fore-, mis-   | con-/com-, re-, pre-, trans-, in-/im-        | tele-, auto-, hyper-, hypo-, anti- |
| **常见后缀**     | -ness, -er, -ing, -ful, -ish, -ly | -tion/-sion, -ment, -ance/-ence, -ity, -able | -logy, -phobia, -ism, -ist, -osis  |

| 转换类型                         | 常见后缀                                    | 含义/用法                                            | 示例                                                                                                |
| ---------------------------- | --------------------------------------- | ------------------------------------------------ | ------------------------------------------------------------------------------------------------- |
| **名词 → 形容词**                 | -ful, -ous, -al, -ic, -y, -ive, -ish    | 表示“具有…特性的、充满…的”                                  | beauty → beautiful (美丽的) danger → dangerous (危险的) nation → national (国家的) child → childish (孩子气的) |
| **动词 → 形容词**                 | -able/-ible, -ive, -ant/-ent, -ed, -ing | -able: “可…的” -ive: “有…倾向的” -ed/-ing: 过去/现在分词作形容词 | read → readable (可读的) attract → attractive (吸引人的) bore → bored/boring (无聊的/令人无聊的)                 |
| **形容词 → 动词**                 | -en, -ify, -ize/-ise                    | -en: “使…变得” -ify/-ize: “使…化”                     | weak → weaken (使变弱) pure → purify (净化) modern → modernize (现代化)                                   |
| **名词 → 动词**                  | -en, -ify, -ize/-ise, -ate              | 表示“使成为…、进行…动作”                                   | heart → hearten (鼓励) terror → terrorize (恐吓) vaccination → vaccinate (接种疫苗)                       |
| **形容词/名词 → 副词**              | -ly                                     | 最常见，表示“以…方式”                                     | quick → quickly (快速地) happy → happily (快乐地)                                                       |
| **名词 → 形容词（表示“像…的”或“属于…的”）** | -like, -ly, -ese, -ian                  | 表示相似、所属、国籍等                                      | child → childlike (像孩子的) friend → friendly (友好的) China → Chinese (中国的) music → musician (音乐的/音乐家) |
| **其他：表示“小/爱称/贬义”**           | -y/-ie, -let, -ling                     | -y: 小型、可爱 -let: 小东西                              | dog → doggy (小狗) book → booklet (小册子) duck → duckling (小鸭)                                        |

|后缀|含义|常见用法（形容词 → 名词）|示例|
|---|---|---|---|
|**-ness**|表示“性质、状态、程度”|最常见、最通用，常加在以-y、-ful 等结尾的形容词后|happy → happiness (幸福) kind → kindness (善良) dark → darkness (黑暗) sad → sadness (悲伤)|
|**-ity**|表示“性质、状态、品质”|多加在以-ble、-ic 等结尾的形容词后，常用于正式或拉丁来源词|possible → possibility (可能性) pure → purity (纯洁) able → ability (能力) real → reality (现实)|
|**-cy**|表示“状态、品质、地位”|常加在以-ent、-ant 等结尾的形容词后|frequent → frequency (频率) accurate → accuracy (准确性) private → privacy (隐私) urgent → urgency (紧急)|
|**-ance / -ence**|表示“行为、状态、过程”|-ance 多用于以-ant 结尾；-ence 用于以-ent 结尾|important → importance (重要性) tolerant → tolerance (容忍) different → difference (差异) patient → patience (耐心)|
|**-hood**|表示“状态、身份、时期”|常表示集体或生命周期状态|child → childhood (童年) adult → adulthood (成年) brother → brotherhood (兄弟情谊) false → falsehood (虚假)|
|**-ism**|表示“主义、信念、状态”|多用于表示意识形态或特性|real → realism (现实主义) hero → heroism (英雄主义) modern → modernism (现代主义)|
|**-ship**|表示“状态、技能、关系”|常表示职位或关系状态|friend → friendship (友谊) leader → leadership (领导力) hard → hardship (艰难)|
|**-th**|表示“状态、品质、结果”|较古老的后缀，常有拼写变化|warm → warmth (温暖) true → truth (真相) wide → width (宽度) grow → growth (成长)|

| 后缀                         | 含义               | 常见用法（动词 → 名词）                     | 示例                                                                                                  |
| -------------------------- | ---------------- | --------------------------------- | --------------------------------------------------------------------------------------------------- |
| **-tion / -sion / -ation** | 表示“动作、过程、结果”     | 最常见，拉丁来源动词，多用于正式语体                | educate → education (教育) decide → decision (决定) create → creation (创造) expand → expansion (扩张)      |
| **-ment**                  | 表示“动作、结果、状态”     | 常加在以-en、-ify 等动词后                 | move → movement (运动) develop → development (发展) achieve → achievement (成就) excite → excitement (兴奋) |
| **-ance / -ence**          | 表示“动作、状态、行为”     | -ance 多用于以-ain、-ant；-ence 用于以-ent | perform → performance (表演) accept → acceptance (接受) exist → existence (存在) depend → dependence (依赖) |
| **-al**                    | 表示“动作、结果”        | 常与 -tion 搭配，或单独使用                 | arrive → arrival (到达) refuse → refusal (拒绝) propose → proposal (提议)                                 |
| **-ing**                   | 表示“动作、过程、活动”或动名词 | 最通用，可表示持续动作或结果                    | run → running (跑步) sing → singing (唱歌) build → building (建筑物/建造)                                    |
| **-ure**                   | 表示“动作、结果”        | 多用于以-t 或 -s 结尾的动词                 | fail → failure (失败) expose → exposure (暴露) close → closure (关闭)                                     |
| <br>**-ery / -ary**        | 表示“行为、行业、集合”     | -ery 多表示行为或场所                     | rob → robbery (抢劫) deliver → delivery (递送) bound → boundary (边界)                                    |
